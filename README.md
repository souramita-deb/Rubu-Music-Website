## 🎧 Rubu – Music Web Player

Rubu is a browser-based music player built using HTML, CSS, and Vanilla JavaScript.  
This project was created after learning JavaScript concepts to apply them in a practical way.

---

## Features

- Album-based music organization  
- Play / Pause functionality  
- Previous & Next controls  
- Automatic next song playback  
- Interactive seek bar with time display  
- Volume control  
- Dynamic sidebar song list  
- Responsive layout  

---

## Tech Stack

- HTML5  
- CSS3  
- Vanilla JavaScript  
- Fetch API  
- Audio API  

---

## Project Structure

```
RUBU-MUSICWEB/
│
├── index.html
├── script.js
├── style.css
├── utility.css
│
├── images/
└── songs/
    ├── AlbumName/
        ├── mp3 files
        ├── cover image
        └── info.json
```

Each album folder contains its own songs, cover image, and a configuration file (`info.json`) used to display album details.
